# ALUGA FÁCIL - Projeto Organizado

Estrutura organizada em duas partes:

- `frontend/`: páginas HTML estáticas, CSS, JS, imagens e dados temporários.
- `backend/`: API Laravel para evoluir o sistema de aluguel de carros.

## Como abrir o frontend

Abra a pasta `frontend` no VS Code e use a extensão Live Server no arquivo `index.html`.

## Páginas principais

- `frontend/index.html` - Home
- `frontend/pages/login.html` - Login
- `frontend/pages/cadastro.html` - Cadastro
- `frontend/pages/recuperar-senha.html` - Recuperar senha
- `frontend/pages/veiculos.html` - Catálogo de veículos
- `frontend/pages/veiculo-detalhe.html` - Detalhe do veículo
- `frontend/pages/orcamento.html` - Solicitar orçamento
- `frontend/pages/dashboard-cliente.html` - Painel do cliente
- `frontend/pages/dashboard-admin.html` - Painel administrativo
- `frontend/pages/cadastro-veiculo.html` - Cadastro de veículo
- `frontend/pages/gestao-veiculos.html` - Gestão de veículos
- `frontend/pages/clientes.html` - Gestão de clientes
- `frontend/pages/reservas.html` - Gestão de reservas

## Próximo passo

Após ajustar o visual, o próximo passo é conectar as páginas ao backend Laravel usando `frontend/assets/js/api.js`.
